## Version 7.0-1 (2026-02-15)

### Initial Release

**Features:**
- Multiple OpenAI model selection (GPT-4o, GPT-4o Mini, GPT-4 Turbo, GPT-4, GPT-3.5 Turbo, O1, O1 Mini)
- Real-time streaming responses
- Conversation history with localStorage persistence
- Custom system prompts for AI behavior customization
- Adjustable temperature, top_p, and max_tokens parameters
- Code syntax highlighting with copy buttons
- Full markdown rendering support
- Custom API endpoint configuration for OpenAI-compatible services
- Clear conversation history functionality
- Modern gradient UI with smooth animations
- Dark mode code blocks
- Responsive layout

**Technical:**
- Compatible with Zabbix 7.0+
- PHP 8.0+ support
- Built on Zabbix widget framework
- Client-side conversation storage
- Abort controller for stopping responses
